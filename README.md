# 🚀 Smart Age Calculator

A Python-based Smart Age Calculator that calculates a user's age in **years, months, and days** based on their birth date.

This project was built as my first programming project while learning Python and software engineering fundamentals.

---

## 📌 Features

- Calculates age in:
  - Years
  - Months
  - Days
- Accepts birth month as:
  - Month name (e.g., January)
  - Month number (e.g., 1)
- Validates user input
- Prevents future birth dates
- Handles leap years
- Uses modular programming with functions

## ▶️ How to Run

Clone the repository:

```bash

```

Navigate into the project:

```bash
cd smart-age-calculator
```

Run:

```bash
python age_cal.py
```
